# 競馬点数計算機

競馬のBOX買いの点数と合計金額を自動計算するWebアプリケーションです。

## 機能

- **BOX点数表**: 選択した頭数に基づいて、各買い目のBOX点数を表示
- **早見表**: 頭数別のBOX点数を一覧表示
- **フォーメーション計算**: 選択した馬番に基づいて点数を計算

## 技術スタック

- Astro
- TypeScript
- Tailwind CSS

## セットアップ

```bash
# 依存関係のインストール
bun install

# 開発サーバーの起動
bun run dev

# ビルド
bun run build

# プレビュー
bun run preview
```
